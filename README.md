# pygeoapi-indg-20-60

## Description

pygeoapi demo instance for the indg 20-60 project

## Setup

1. Run `git clone https://github.com/maxcollombin/pygeoapi-indg-20-60.git` to clone this repo    
2. Run `docker-compose up -d` to start the composition
